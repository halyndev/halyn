# Contributing to Halyn

## How to contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes with proper BSL-1.1 headers on every new file
4. Add tests if applicable
5. Open a pull request

## License

By contributing, you agree your contributions will be licensed under BSL-1.1.
